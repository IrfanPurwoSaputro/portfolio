Sistem Informasi Data Pengusahaan

Sistem Informasi Data Pengusahaan adalah sebuah aplikasi berbasis website yang digunakan untuk mengunggah laporan data pengusahaan per bulan dari setiap rayon. Aplikasi tersebut saya buat dengan teman saya selama masa PKL di PT. PLN (Persero) UP3 Madiun. Dalam hal ini saya dan teman saya ditempatkan di bagian pengadaan. Pada saat itu saya dan teman saya diberikan tugas untuk membuat sistem yang digunakan untuk menyimpan hasil laporan pengadaan dari perusahaan yang saat itu harus tidak terstruktur untuk setiap daerah rayon.

Kemudian kami merancang untuk use case dari aplikasi tersebut. Setelah itu, kami berikan kepada supervisor bagian pengusahaan selaku pemberi tugas kami untuk menjelaskan alur dari website tersebut. Kemudian kami mulai melakukan pengembangan aplikasi tersebut menggunakan Boostrap untuk tampilan dan bahasa pemrograman PHP agar proses pada website tersebut dapat bekerja. Laporan yang sudah diunggah kami simpan ke dalam database mySQL.

Proses yang kami rancang dalam aplikasi tersebut yaitu berupa pengunggahan laporan atau file. Dalam proses tersebut terdapat 2 user yaitu admin dan user. Admin selaku supervisor bagian perencanaan bertugas untuk mengunggah laporan dengan cara
1. Memilih kode rayon sesuai laporan
2. Memilih bulan dan tahun
3. Mengunggah laporan dan memilih button simpan
Kemudian untuk user selaku petugas dalam setiap rayon dapat melihat laporan sesuai rayon tersebut dan juga dapat mencari laporan dari bulan dan tahun di dalam pilihan form dan dropdown yg tersedian di tampilan tersebut.
